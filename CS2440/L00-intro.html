Course Overview

  • Upon completion of this course, the student should be able to...
    1. identify errors in programs and debug them
    2. construct Java programs using object-oriented programming
    3. understand the building blocks of data structures, arrays and linked lists
    4. analyze and compare the time- and space-complexity of algorithms
    5. write programs and solve problems using recursion
    6. compare a variety of sorting and searching algorithms
    7. demonstrate a basic proficiency with Linux/UNIX systems
