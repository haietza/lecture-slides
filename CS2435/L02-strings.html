Strings

  • Strings are sequences of characters
  • Interact with data using square bracket notation
  • We refer to this method as "indexing" a sequence
  • Sequences in Python start counting at zero
    • 							
								x = "abcdefg" # 7 letters
								print(x[1])   # what will this print?
								              # what is the index of the last character?

Strings

  • To determine the length of a string, use the len() function, which returns an integer
    • 							
								x = "hello world" 
								print(len(x)) # what will len(...) return?

Strings

  • If we want to access the last character of the string, we can use the len() function to help
    • 							
								x = "abcxyz" 
								print(x[len(x)-1]) # len(x)-1 is the index of the last character
  • Accessing values at the end of a sequence is so common that Python provides a shorthand syntax for it: negative indices
    • 							
								x = "abcxyz" # -1 is the last element, -2 next to last, etc
								print(x[-3]) # what does this print?

Strings

  • Another common operation is slicing, retrieving a substring from a string
  • This does not modify the original string!
  • Basic slicing syntax: x[start:end] (start inclusive, end exclusive)
    • 							
								x = "the quick brown fox jumps over the lazy dog" 
								print(x[4:9]) # this will print "quick"
  • Negative indexing can be used with slicing

Strings

  • It is common, when slicing, to start at the beginning of the string or to stop at the end of the string
  • Python does the "smart" thing if you omit either index in the slice
    • 							
								x = "the quick brown fox jumps over the lazy dog"
								print(x[0:3]) # this prints "the", the first word
								print(x[:3])  # so does this, first index omitted implies 0
								
								print(x[-3:len(x)]) # this prints "dog", the last word
								print(x[-3:])       # so does this

Strings

  • There are many string methods
						
							x = "hello" + " world"   # + can concatenate strings
							print(len(x))     # this prints 11, the length of the string
							print(x.upper())  # converts the string to uppercase
							
							print('ell' in x) # returns True if substring in string
							print(x.find("ell")) # where does substring appear in str

User Input

  • Programs can pause the program to wait for user input with the input() function
  • You can provide a prompt as an optional argument
						
							x = input()  # the program will wait here until the user responds

							# prompt the user for input again, this time with a proper prompt
							# note the space at the end of the string
							x = input("Please type your name: ")

							# whatever the user types will be stored in the variable `x`

User Input

  • Be careful: input() returns a string
						
							x = input("Type a number: ")
							print(x + 10)  # this is an error; x is a string

User Input

  • To fix this, convert the type!
						
							x = input("Type a number: ")
							x = int(x)                          # can do this in two lines
						
					
						
							x = int(input("Type a number: "))   # or just one
						
					
						
							# don't think you need to convert every single value
							# input() returns a string, no need to convert to string
							x = str(input("Please type your name: ")) # wrong!
