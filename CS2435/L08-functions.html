Functions and Classes

Functions

  • Functions help refactor code by putting commonly repeated operations into reusable blocks
  • They are created using the def keyword, followed by the function name and a parameter list in parentheses
  • Function names follow identifier rules:
    • alphanumeric characters and underscores
    • first character not a digit
    • conventionally not starting with a capital letter

Defining and Calling Functions

						
						def say_hello():
							print("Hello World")

Call the function: say_hello()

Parameters and Arguments

  • Parameters are variables that a function receives as input
  • Arguments are the values passed into the function when it's called
						
						def say_hello_to(name):
							print("Hello, %s!" % name)

						say_hello_to("Sarah") # "Sarah" is the argument for the 'name' parameter

Return Statements

  • return statements instantly terminate the function and return control to the calling context
  • They can optionally return a value through the function call
						
						def sum_squares(numbers):
							a = 0
							for x in numbers:
								a += x * x
							return a # Returns the sum of squares

Object-Oriented Programming

  • Methods are class-owned functions that operate on the data within an object
  • Accessors: Methods designed to get information from an object (e.g., str.split())
  • Mutators: Methods designed to change the internal data of an object (e.g., list.append())

Classes in Python

  • A blueprint that describes the structure of an object
  • Created using the class keyword
  • Class names conventionally start with a capital letter
						
					class Student:
						pass # An empty class

The self Parameter

  • All class methods require self as their first parameter
  • self is a reference to the object the method is called on
  • Python automatically provides self when the method is called

Constructors

  • A method called when an object is created in order to set up its data
  • Must be named __init__
						
					class Rectangle:
						def __init__(self, width, height):
							self.width = width
							self.height = height

When calling the constructor Rectangle(3, 5), self is automatically passed; only arguments for width and height are provided

String Representation

  • Special method for converting object to a string representation
  • Must be named __str__
  • Does not print anything, simply returns string to be printed
						
					class Rectangle:
						# ... (init method) ...
						def __str__(self):
							return "Rectangle(%d x %d)" % (self.width, self.height)

Printing an object print(r) will automatically call its __str__ method if defined
