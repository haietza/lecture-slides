Print Statements

  • We use the print statement to print data to the screen. This function is provided by Python
  • The print stmt always adds a newline at the end
  • The following examples show different types of data being printed. We'll talk more about that
						
							print("Hello World") # prints: Hello World
							print(42)            # prints: 42
							print(1 + 2 + 3)     # prints: 6

Data Types

  • There are many types of data that Python can store:
						
							print(type(42))            # <class 'int'>
							print(type(3.141592))      # <class 'float'>
							print(type("Hello World")) # <class 'str'>
							print(type(True))          # <class 'bool'>
							print(type(None))          # <class 'NoneType'>

Data Types — int & float

  • These are numeric types: integers and floating-point numbers (decimals)
  • ints can only hold whole numbers (positive or negative), but have perfect precision
  • Unlike other languages, there is no maximum value that Python can hold in an integer

Data Types — int & float

  • floats can hold decimals, but may be subject to rounding errors, due to the way they are stored
  • floats may be written in scientific notation
  • ints and floats can be used interchangeably, but converting between types has its own rules
						
							print(1 + 2)     # prints: 3      - result is int
							print(1 + 2.5)   # prints: 3.5    - result is float
							print(0.1 + 0.2) # prints what?   - result is float
							print(2e2)       # prints what?   - scientific notation

Data Types — str

  • strs (strings) hold text data
  • Strings can be delimited (started and ended) with double-quotes (") or single-quotes (')
						
							print("Hello World")       # prints: Hello World
							print('Welcome to CS2435') # prints: Welcome to CS2435
							
							print("42")                # prints the string containing 42
							                           # this is NOT the number 42

Variables

  • Variables are named locations in memory where data can be stored
  • The name of a variable is subject to certain rules:
    • Made of only letters, numbers, and underscores
    • First character cannot be a number
    • Identifiers may not contain spaces
						
							a = 42             # contains the integer 42
							b = "Hello World"  # contains the string "Hello World"
							c = "42"           # contains the string "42", not a number

More on Print Statements

  • Sometimes we want to combine data for printing
  • Method #1: print() can take multiple arguments
						
							dogs = 2
							cats = 3
							print("I have", dogs, "dogs and", cats, "cats")
							# prints "I have 2 dogs and 3 cats"
							# (note the added spaces)

More on Print Statements

  • Method #2: String concatenation
						
							dogs = 2

							# WRONG! this will produce an error. cannot add string and number
							print("I have" + dogs + "dogs")

							# better, but if we don't add spaces, this won't look right
							print("I have" + str(dogs) + "dogs")

							# correct. note the extra spaces that we typed manually
							print("I have " + str(dogs) + " dogs")

More on Print Statements

  • Method #3: Format strings
    • 							
								# format specifier "%d" is for integers
								# % plugs values from right side into placeholders on the left
								dogs = 2
								print("I have %d dogs" % (dogs))

								temp = 31.41592
								print("The temperature is %8.2f C" % (temp))
								# prints "The temperature is    31.42 F"
  • Format strings give a lot of control over the presentation of data
