Loops

Loop Basics

  • We want a block of code to run repeatedly based on some condition

while Loop

  • A while loop runs an indented block of code, or the loop body, repeatedly as long as the loop condition is true
    • 							
								x = 0
								while x < 10:
									print("Hello world")

Infinite Loops

  • There are some programs we don't want to stop
  • But how do we prevent infinite loops when they are not what we intended?
Condition of while loop is x < 10
For loop to stop, x has to be x has to be >= 10
We can change a variable's value and resave it into the variable
x + 1 does not store the result, but x = x + 1 does (not mathematical, it's an assignment statement)
Every time loop is executed, x gets incremented
We can print out x so we can see how it changes

x value is loop control variable
We don't want to do too much to the loop control variable, because it makes it complicated to know how many times loop will run
x = x + 1 is common construction
Python supports syntax for this

for Loop

  • We have a collection of values
  • We want a program to iterate, or go through those values
    • 							
								for elem in "abcdefg":
									print(elem)

Main way to use for loop is probably operating on a range of values, and we use range method for that
If interested in looping a specific number of times (not on what you are looping over), range function is good for that

The range Function

  • Generates a range of values the for loop can iterate over
    • 							
								for i in range(10):
									print(i)
  • Starts at 0
  • Goes up to, not including, 10
Single argument, start at 0, go up to not including value (goes from 0 to 4)
Two arguments, specify start and endpoint (starts at 1, goes up to 4, have to do 1,11 to get 1-10 as values)

The range Function: Two Arguments

    							
								for i in range(1,10):
									print(i)
  • Starts at 1
  • Goes up to, not including, 10

Third value is step value/size (increment by 2 each time)
Try to adopt meaningful variable names, x is not meaningful
Text we are iterating over, elem is element in text
i is not a meaningful variable name, but this is exception; we use i, j, and k usually for loop control variables

The range Function: Three Arguments

    							
								for i in range(1,10,2):
									print(i)
  • Starts at 1
  • Goes up to, not including, 10
  • step size of 2 between values
  • Generates the sequence [1,3,5,7,9]
Possible to nest these structures inside one another
Accomplish same thing another way (less efficient) - if i not divisible by 2 (odd numbers / 2 have remainder of 1)
While elem is A, it goes through inner loop for 0, 1, and 2
Then it goes to the outer loop and elem is B, then goes through inner loop again, and so on

Nesting Loops

  • Alternative solution, less efficient:
    • 							
								for i in range(10):
									if i % 2 == 1:	
										print(i)
  • Common for multi-dimensional data
    • 								
									for elem in "ABCD":
										for i in range(3):
											print(elem, i)

while loop lets us say when we want to stop doing the loop
But condition is only checked at top of loop
Sometimes we are in the middle of doing something and we want to terminate immediately
break and continue are used for this

break and continue

  • Flow of a loop is typically controlled by the condition
  • Sometimes we want to disrupt that flow
break keyword breaks out of closest possible loop
If loop nested inside of loop, breaks out of innermost loop
Even though range goes to 1000, we stopped after 10
In this simple example, we could probably address limiting the iterations in the header of the loop, break is typically used in more complex code
Sometimes considered impropertly terminating a loop, they make it difficult to follow flow of loop

The break Statement

  • Immediately terminate a loop
  • Applies to closest, or innermost, loop
    • 							
								for i in range(1000):
									if i == 10:
										break
									print(i)

Sometimes we don't want to terminate loop, but skip values (skip odd numbers)
Skips the remainder of the loop every time it is odd number; goes back to top of loop

The continue Statement

  • Current iteration of innermost loop is terminated
  • But the loop continues
    • 							
								for i in range(25):
									if i % 2 == 1:
										contnue
									print(i)
